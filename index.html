Skill management
for developers

Document and present skills for individuals and teams. The devradar editor allows you to:

  • create your own techradar
  • customize adoption levels
  • define quadrant categories
  • create radar blips via GUI
  • view all skills in a list
  • search for specific skills
  • import/export via .toml files
  • share your devradar via URL

Setup options

Serverless

Host your own instance on Google Firebase with integrated authentication, user management and web-based blip editing.

Setup Guide

Editor App

View and edit your devradar on our client-only web application. No data is sent to the server, everything stays on your device. You can download your devradar as TOML file.

Open editor

Docker Server

Run devradar in a Docker container to let others browse your skills. Requires a TOML file created via the editor web app.
